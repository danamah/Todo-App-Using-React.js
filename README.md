# Todo App – React + Vite

A clean, fully functional, and responsive **Todo App** built with React.js, completing the famous Frontend Mentor "Todo app" challenge with extra polish and modern features.

![Todo App Preview](./public/preview.jpg)
*(Desktop + Mobile – Dark & Light modes)*

## Features

- Add new todos by typing and pressing **Enter**  
- Toggle todo completion (check/uncheck)  
- Mark all tasks as complete  
- Delete individual todos  
- Clear all completed todos  
- Filter tasks: **All / Active / Completed** (with active state styling)  
- Live counters for completed & remaining items  
- Full **Dark / Light** **Light** mode toggle (sun/moon icon)  
- Fully responsive (mobile-first)  
- Clean, readable, and modern React code  

## Tech Stack

- **React 18** + **Vite**  
- JavaScript (ES6+)  
- Custom CSS (no frameworks)  
- React Hooks (`useState`, `useEffect`)  
- Lifting State Up pattern
